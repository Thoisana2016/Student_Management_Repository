# Student Management System

A simple student management system built with Spring Boot.

## Features

- Add/Create a new student
- Edit/Update existing student details
- Delete a student
- View all students

## Data Fields

Each student record includes:
- `id`
- `firstName`
- `lastName`
- `email`

## Technologies Used

- Java(17)
- Spring Boot
- Spring MVC
- Thymeleaf (if using front-end)
- H2 / MySQL Database
